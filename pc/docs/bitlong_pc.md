# bitlong_pc

## build

- command line

```shell
.\build.pc.bat
```

- powershell

```shell
.\build.pc.ps1
```

## StartLitd

- The path `path` is the directory where the config.txt is located.
- `dirpath` is the file storage path of the litd configured in the `config.txt`.
- litd's configuration file is `{{dirpath}}/.lit/lit.conf`.
